# Reference Site Course Card Specifications
## From: https://topskillspk.lovable.app/

### Card Structure (EXACT from reference):
- Border radius: `1.125rem` (18px)
- Padding: `1.25rem` (20px) - `p-5` in Tailwind
- Border: `1px solid #E5E7EB`
- Shadow: `0 1px 3px 0 rgba(0,0,0,0.06), 0 1px 2px -1px rgba(0,0,0,0.06)`
- Hover shadow: `0 8px 12px -2px rgba(0,0,0,0.1), 0 4px 6px -2px rgba(0,0,0,0.08)`
- Hover transform: `translateY(-0.5px)` (very subtle)

### Image:
- Aspect ratio: `aspect-video` (16:9)
- Object fit: `cover`
- Border radius: Top corners only (inherited from card)
- Background: `#048181` (if no image)

### Typography (EXACT from reference):
- Title: `text-base sm:text-lg md:text-xl` (16px/18px/20px), `font-semibold`, `text-[#1F2937]`
- Instructor: `text-sm`, `text-[#6B7280]`
- Description: `text-sm`, `text-[#6B7280]`, `line-clamp-2`
- Price: `text-lg`, `font-bold`, `text-[#048181]`
- Badge: `text-xs`, `font-semibold`, `px-3 py-1`, `rounded-lg`

### Spacing (EXACT from reference):
- Card padding: `p-5` (1.25rem / 20px)
- Content padding: `pt-4` (1rem / 16px) after image
- Title margin: `mb-1.5` (0.375rem / 6px)
- Instructor margin: `mb-2` (0.5rem / 8px)
- Description margin: `mb-3` (0.75rem / 12px)
- Metadata margin: `mb-3` (0.75rem / 12px)
- Price section: `pt-3` (0.75rem / 12px) with border-top

### Colors (EXACT from reference):
- Primary: `#048181`
- Text dark: `#1F2937`
- Text muted: `#6B7280`
- Border: `#E5E7EB`
- Badge (Online): `#048181` background, white text
- Badge (Physical): `#5a9c7d` background, white text

### Font Family:
- `CameraPlainVariable` or fallback to system fonts
